# DevOps CI Mini Project

This project demonstrates a simple Continuous Integration (CI) pipeline using GitHub Actions.

## What this project does
- Runs automated tests on every push
- Validates code changes automatically
- Shows pass/fail status using GitHub Actions

## Tech used
- GitHub
- GitHub Actions
- Python
- Pytest

## Why this project
This project helps understand the basics of DevOps and CI by automating testing without manual effort.

## Outcome
- Automated CI pipeline
- Faster feedback on code changes
- Improved code reliability
